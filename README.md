# GIANTS: Generative Insight Anticipation from Scientific Literature

This repository contains the source code for the project page of **GIANTS: Generative Insight Anticipation from Scientific Literature**.

## Overview

GIANTS is a benchmark and RL-trained model for anticipating scientific insights from parent papers. This website serves as the main project page.

* **arXiv:** [PDF](static/pdfs/insights__arXiv_.pdf)
* **Dataset & Models:** [Hugging Face](https://huggingface.co/giants2026)
* **Code:** [GitHub](https://github.com/)

## Local Development

To preview the website locally, you can use any basic web server. For example, if you have Python 3 installed, you can simply run:

```bash
python3 -m http.server 8000
```

Then, open your browser and navigate to `http://localhost:8000`.
